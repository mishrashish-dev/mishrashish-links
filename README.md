# mishrashish.com /links

This is the active build folder for the first version of the `mishrashish.com/links` one-page website.

## Current structure

- `index.html`
  - redirects root traffic to `/links/`
- `links/index.html`
  - the actual one-page links hub
- `assets/styles.css`
  - page styling
- `assets/script.js`
  - lightweight entrance animation

## Current goal

Build a simple, clean, static page that can later be hosted on GitHub Pages and pointed to the custom domain path `/links`.
